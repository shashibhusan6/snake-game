# 🐍 Snake Game

A classic Snake Game built using **HTML**, **CSS**, and **JavaScript**. Control the snake, collect food to grow longer, and try to achieve the highest score without colliding with the walls or yourself.

---

## 🚀 Features

- 🎮 Classic Snake Game
- 🍎 Random food generation
- 📈 Live score counter
- 🐍 Snake grows after eating food
- 💀 Game Over screen
- 🔄 Restart game functionality
- 🎨 Simple and clean user interface

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🎮 Controls

| Key | Action |
|------|--------|
| ⬆️ | Move Up |
| ⬇️ | Move Down |
| ⬅️ | Move Left |
| ➡️ | Move Right |
